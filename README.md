# Instagram Clone

[Live demo](https://instagram-clone-alkimcaner.vercel.app/)

Built with:

- Next.js
- NextAuth.js
- React
- Typescript
- Firebase
- Tailwindcss
- Headless UI

## Project Setup

### Install dependencies

```bash
yarn install
```

### Start Next.js in development mode

```bash
yarn dev
```

### Build the application for production

```bash
yarn build
```

### Start a Next.js production server

```bash
yarn start
```
