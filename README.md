# Instagram Clone

[Live demo](https://instagram-clone-flawn.vercel.app/)

Built with:

- Next.js
- NextAuth.js
- React
- Typescript
- Firebase
- Tailwindcss
- Headless UI

## Screenshot

![gif](https://github.com/fl4wn/portfolio/blob/main/public/assets/projects/instagram_banner.gif)

## Project Setup

### Install dependencies

```bash
npm install
# or
yarn install
```

### Start dev server

```bash
npm run dev
# or
yarn dev
```

### Build for production

```bash
npm run build
# or
yarn build
```

### Start Node.js server

```bash
npm run start
# or
yarn start
```
